# PrithviNet – Smart Environmental Monitoring Platform

PrithviNet is a full-stack environmental monitoring platform designed to track and analyze **air, water, and noise pollution data** across regions and industries.

The platform provides dashboards, analytics, and AI-assisted insights to help regulators monitor environmental compliance and make data-driven decisions.

---

## Project Overview

This repository contains **two development prototypes** of the PrithviNet system created during the project exploration phase.

### Prototype_1

One implementation of the platform that provides the core features of PrithviNet.

### Prototype_2

An alternate implementation of the same platform with similar features, built using a different approach and structure.

Each prototype contains:

* **Backend** – API services and server logic
* **Frontend (PrithviNet)** – React + Vite dashboard interface
* **docs** – development notes and technical documentation

---

## Features

* Real-time environmental monitoring dashboards
* Industry compliance tracking
* Pollution data visualization
* Geospatial pollution mapping
* AI-assisted environmental insights
* Automated report generation
